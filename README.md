## Rkx's Arch Linux Install Script

```
sudo pacman -Sy

sudo pacman -S git

git clone https://github.com/rackax/rkx_arch --depth=1

cd krushn-arch

# please edit the hostname and username in the post-install.sh 

./install.sh
```
